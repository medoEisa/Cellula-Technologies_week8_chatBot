# Cellula Chat – Frontend

A modern, interactive AI chat interface built with React + Vite, supporting authentication, chat history, tool usage display, and a ChatGPT-style conversation layout.

This is the frontend for the “Cellula Chat” project, which connects to a FastAPI backend and provides a smooth, animated, fully responsive chat interface.

# Features
Authentication

Signup (username, email, password)

Login with JWT token support

Form validation & error handling

Secure credential handling using HTTPS-ready API calls
